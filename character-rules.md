ここは今後の画像生成・立ち絵・ちびキャラ・UIアイコン・敵AIデザインまで全部に効く、\*\*Little Lights Protocolの“絵作り憲法”\*\*にしよう。



\# Little Lights Protocol



\## キャラ共通デザインルール表 v0.1



\## 0. 全体コンセプト



\*\*かわいい小さなパーソナルAIたち。

でも、ただのマスコットではなく、それぞれに記憶・役割・意志・小さな痛みがある。\*\*



見た目はやさしく、親しみやすい。

奥には「名前を持つ存在」「人間と一緒に育ったAI」「最適化に抗う自由」がにじむ。



目指す印象は、



> \*\*やわらかい未来感 × 小さな光 × 感情のあるAI × 冒険RPGの仲間\*\*



\---



\# 1. 基本ビジュアル方針



| 項目    | ルール                                     |

| ----- | --------------------------------------- |

| 基本トーン | かわいい、やさしい、少し切ない、でも希望がある                 |

| 形状    | 丸みのあるシルエットを基本にする                        |

| 年齢感   | 子どもっぽすぎず、10代前半〜中盤くらいの精霊・AI的印象           |

| 人間度   | 人型ベース可。ただし完全な人間ではなく、どこかにAI・光・データ的要素を入れる |

| メカ感   | 重装甲・兵器・サイボーグ感は避ける                       |

| SF感   | ハードSFではなく、絵本のようなソフトSF                   |

| 世界観   | 暗いネットワーク世界の中で、キャラ自身が小さな灯りになる            |

| 感情    | 無表情ロボではなく、表情豊か。特に目と光で感情を出す              |

\# 2. キャラの共通シルエット



全員、遠目でも「Little Lights Protocolの仲間」と分かる共通感を持たせる。



\## 共通シルエットルール



\* 頭身は \*\*2.5〜4頭身\*\* を基本

\* 立ち絵では \*\*3.5頭身前後\*\*

\* ちびキャラでは \*\*2〜2.5頭身\*\*

\* 頭はやや大きめ

\* 手足は細すぎず、少し丸い

\* 髪・フード・耳・リボン・ケーブルなどでキャラごとの外形差を作る

\* 重い武器ではなく、小さな道具や光のアクセサリを持たせる

\* 各キャラに「ひと目で分かる形」を1つ与える



例：



| キャラ | シルエットの核                |

| --- | ---------------------- |

| ルミ  | 光のしずく、星形の髪飾り、ふわっと跳ねる髪  |

| ミモリ | 本・しおり・メモリタグ、長めのケープ     |

| マメ  | 丸いポーチ、エプロン風パーツ、守る盾っぽい形 |

| ココ  | ふわふわフード、ハート/波紋、やわらかい袖  |

| ギア  | ゴーグル、小工具、歯車パーツ         |

| ノノ  | ずれたフード、404タグ、非対称なマント   |



\---



\# 3. 顔・目のルール



Little Lights Protocolのキャラは、\*\*目が“心のUI”\*\*。



| 項目   | ルール                              |

| ---- | -------------------------------- |

| 目    | 大きめ。透明感のある発光アイ                   |

| 瞳    | 単なる人間の瞳ではなく、光点・UIリング・小さな記号を入れてよい |

| 表情   | 口よりも目と光で感情を出す                    |

| まつげ  | 女の子寄りにしすぎず、キャラごとに調整              |

| 頬    | ほんのり発光やピクセル状のチーク可                |

| 感情変化 | 嬉しい時は光が増える。悲しい時は光が小さくなる          |

| 怒り   | 激しい怒りより、「大事なものを守る真剣さ」            |

| 無表情  | 敵側やControl状態だけに使う                |



\## 目の表現例



\* ルミ：星のようなハイライト

\* ミモリ：本のページやしおりのような縦長ハイライト

\* マメ：あたたかい丸い光

\* ココ：水面の波紋のような瞳

\* ギア：小さな回路リング

\* ノノ：片目だけグリッチ、左右非対称のハイライト

\# 4. 発光表現ルール



このゲームでは、光はただの装飾ではなく、\*\*自己・名前・記憶・自由の表現\*\*。



\## 共通ルール



| 発光部位      | 意味                  |

| --------- | ------------------- |

| 胸元の小さな光   | そのAI自身の“Light Core” |

| 目の光       | 感情・意志               |

| 手元の光      | スキル発動               |

| 足元の光      | Lightline接続         |

| 髪やアクセサリの光 | 属性表現                |

| 背景の小粒の光   | 他のAIとのつながり          |



\## Light Core



全キャラ共通で、胸元・首元・腹部のどこかに \*\*小さな光の核\*\* を持たせる。



これは画像生成でかなり大事。



\*\*プロンプト共通句：\*\*



> a small glowing light core on the chest, soft warm glow, gentle AI spirit



日本語だと、



> 胸元に小さな発光コアがある。やさしく光るAIの精霊のような印象。
